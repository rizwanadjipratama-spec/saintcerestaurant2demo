# Project: Saintce Restaurant Demo - Premium Modern Dining

## 🎯 Objectives
Build a high-end, production-grade restaurant demo website using Next.js (App Router) and Tailwind CSS. The site must feel premium, professional, fast, and fully responsive with zero dummy content.

## ⚙️ Technical Stack
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Deployment Ready**: Production-grade architecture
- **Assets**: Next/Image with high-quality Unsplash images

## 🎨 Design System
- **Heading Font**: Cormorant Garamond (Elegant Luxury)
- **Body Font**: Inter (Clean Readability)
- **Colors**:
  - Deep Black: `#0F0F0F`
  - Gold Accent: `#C6A969`
  - Cream: `#F8F5F0`
  - Soft Gray: `#2A2A2A`
- **Aesthetic**: Cinematic, smooth hierarchy, elegant spacing.

## 🧱 Site Structure
1. **Navbar**: Sticky, scroll-aware, mobile-responsive menu with "Book Table" CTA.
2. **Hero**: Full-width cinematic section with high-end storytelling.
3. **About**: Emotional storytelling about the dining experience.
4. **Signature Menu**: Highlight dishes with realistic data.
5. **Full Menu**: Interactive tabs (Appetizer, Main, Dessert, Drinks).
6. **Dining Experience**: Highlighting quality, ambiance, and ingredients.
7. **Gallery**: Masonry grid with hover interactions.
8. **Reservation**: Fully functional UX (simulated success state).
9. **Events**: Private dining and gathering booking section.
10. **Testimonials**: Realistic guest reviews.
11. **FAQ**: Interactive SEO-friendly accordion.
12. **Contact**: Static map, clickable WhatsApp, hours of operation.
13. **Footer**: Socials, secondary navigation, and legal.

## ✨ UX & Quality Requirements
- **Smooth Interaction**: Optimized scrolling, scroll reveal, subtle parallax.
- **Micro-interactions**: Hover effects on cards, images, and buttons.
- **Responsiveness**: Perfect layout on Mobile, Tablet, and Desktop.
- **SEO**: Proper title tags, meta descriptions, and semantic HTML.
- **Performance**: Lighthouse optimized, 0 console errors, 0 broken links.

## ❗ Constraints
- **NO DUMMY LINKS**: All buttons must lead somewhere or trigger an action.
- **NO 404s**: Every route/section must be complete.
- **NO PLACEHOLDERS**: Use real high-quality images and copy.
